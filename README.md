# Senhas-Seguras

SecurePass – Gerador de Senhas Seguras

O SecurePass é uma aplicação web desenvolvida com HTML, CSS e JavaScript para gerar senhas fortes e seguras de forma rápida e intuitiva. O usuário pode personalizar o tamanho da senha e escolher incluir letras maiúsculas, letras minúsculas, números e símbolos. Além disso, o projeto conta com um indicador visual da força da senha e um botão para copiar a senha gerada para a área de transferência.

Este projeto foi desenvolvido com foco em praticar conceitos de desenvolvimento front-end, manipulação do DOM e criação de interfaces responsivas, além de promover boas práticas de segurança digital na criação de senhas.
